# 静态 HTML 页面说明

这是用于课程提交的静态页面原型，不依赖前后端服务。

## 打开方式

1. 进入 `static-html` 目录
2. 双击 `login.html` 或 `index.html`
3. 浏览器中即可查看页面

## 文件说明

- `login.html`：登录页原型
- `index.html`：首页概览原型
- `appointments.html`：挂号预约页原型
- `triage.html`：AI 分诊页原型
- `assets/style.css`：统一样式
- `assets/app.js`：简单演示交互（仅前端模拟）

## 备注

- 页面中的提交、分诊结果是前端演示，不会写入数据库。
- 若需真实业务功能，请运行项目主系统（`02-日常启动-本机和校内访问.bat`）。
